# TDDICOM
A TouchDesigner component that loads DICOM image files. Based around the PYDICOM project.

https://pydicom.github.io

## Dependencies

* Latest Experimental Version of TouchDesigner
* Python 3.7.2
* pillow
* pylibjpeg
* pylibjpeg-libjpeg
* pylibjpeg-openjpeg
